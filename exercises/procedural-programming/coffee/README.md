# Coffee Procedural Programming Exercise

## Overview
This exercise demonstrates how a real-world process (making coffee) can be broken down into a procedural algorithm and implemented in Python.

## Concepts Used
- Functions
- Loops (while, for)
- Conditionals (if/elif/match)
- User input handling
- Control flow

## What I Practiced
- Translating an algorithm into code
- Structuring logic into functions
- Debugging indentation and scope issues
- Managing execution flow between steps

## Key Takeaway
An algorithm defines the steps.  
The program executes those steps in a structured way.

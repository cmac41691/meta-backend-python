# Loops and Advanced Conditionals

## Control Flow Review
- if / elif / else controls program decisions
- Conditions evaluate to True or False

## Match Statement
- Used for fixed value comparisons
- Cleaner than many elif blocks in some cases

## Loops

### while loop
- Repeats while condition is True

### for loop
- Iterates over a sequence

## Key Takeaways
- Loops allow repeated execution
- Validation often requires repetition
- Conditionals decide which path runs

## Backend Thinking
- Retry systems
- Input validation
- Menu systems
- Processing repeated tasks

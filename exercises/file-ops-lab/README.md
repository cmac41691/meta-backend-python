# File Operations Lab

## Overview
This exercise practices file handling in Python by reading from a file, writing to a file, and storing file contents in data structures.

## Goals
- Read entire file contents
- Read file contents into a list
- Write the first line of content into a new file
- Return even-numbered lines
- Return file contents in reverse order

## Concepts Used
- functions
- file handling
- lists
- loops
- string processing

## Backend Thinking
Input file → read data → process data → return or write output
